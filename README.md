# Steveify

A Spotify client built with React.

## Features

Here's a quick overview of how Steveify will function:

* As a user, I should be able to search for tracks.
* As a user, I should be able to view a list of tracks returned by the Spotify API.
* As a user, I should be able to create a custom playlist.
* As a user, I should be able to add tracks to a custom playlist.
* As a user, I should be able to save a custom playlist and its tracks to my Spotify account.
